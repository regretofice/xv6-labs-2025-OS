# xv6 操作系统学习实验 (同济大学 2026 暑期课程)

本项目为 **同济大学 2026 学年度暑期课程《操作系统》** 的个人实验仓库，基于 MIT 的 `xv6-labs-2025` 进行学习和实践。

##  课程信息
- **课程名称**：操作系统 (Operating Systems)
- **开课单位**：同济大学
- **学年学期**：2026 年暑期学期
- **课程主页**：[xv6-labs-2025](https://pdos.csail.mit.edu/6.828/2025/)

### 环境要求
- 操作系统：Windows WSL2 + Ubuntu24.04
- 工具链：RISC-V 交叉编译器 (`riscv64-unknown-elf-gcc`)、QEMU (模拟器)
- 构建工具：`make`, `git`
